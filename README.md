## ğŸ› ï¸ Customization

### Update Personal Information

1. **Name & Title**: Edit the hero section in `index.html`
2. **About Me**: Update the about section content
3. **Skills**: Add/remove skill cards in the skills section
4. **Work Experience**: Modify work cards with your experience
5. **Projects**: Add your projects with images and descriptions
6. **Contact Info**: Update email, phone, and social media links

### Color Scheme

All colors are defined as CSS variables in `style.css`:

```css
:root {
    --primary-purple: #8B5CF6;
    --dark-purple: #6D28D9;
    --light-purple: #A78BFA;
    --neon-purple: #C084FC;
    --deep-purple: #5B21B6;
}
```

### Add Project Images

Place your project screenshots in the `assets/` folder and update image sources in `index.html`:

```html
<img src="assets/your-project.jpg" alt="Project Name">
```
